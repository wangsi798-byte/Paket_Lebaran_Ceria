# SiPaLe v1.0.0 - Rilis Perdana

## 🎉 Selamat Datang di SiPaLe!

### Apa yang Baru?
- Sistem tabungan paket Lebaran terintegrasi
- Manajemen peserta multi-role
- Distribusi paket fleksibel
- Keamanan berlapis
- Notifikasi cerdas

### Fitur Utama
1. **Manajemen Peserta**
   - Registrasi mudah via OTP
   - Pilih paket (uang/sembako)
   - Atur target tabungan

2. **Setoran Fleksibel**
   - Periode nabung 1 tahun
   - Setoran harian/mingguan/bulanan
   - Pantau progress real-time

3. **Distribusi Paket**
   - Pilih paket sesuai preferensi
   - Stok paket terkontrol
   - Bukti pengambilan digital

4. **Keamanan Canggih**
   - Autentikasi OTP
   - Enkripsi data
   - Audit log

5. **Notifikasi Cerdas**
   - WhatsApp
   - SMS
   - Email
   - Pengingat target

### Teknologi
- Backend: Node.js, Express
- Database: MongoDB
- Frontend: React
- Deployment: Docker
- Hosting: Cloud-ready

### Panduan Memulai
1. Daftar sebagai peserta
2. Atur target tabungan
3. Mulai menabung
4. Pantau progress
5. Terima paket Lebaran

### Tips
- Konsisten dalam menabung
- Manfaatkan fitur notifikasi
- Pilih paket sesuai kebutuhan
