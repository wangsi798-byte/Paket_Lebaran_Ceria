# 📘 Panduan Pengguna SiPaLe

## 🎯 Tujuan Aplikasi
SiPaLe membantu warga kampung menabung untuk paket Lebaran dengan mudah, aman, dan transparan.

## 👥 Role Pengguna

### 1. Peserta
#### Proses Bergabung
1. Daftar via nomor HP
2. Verifikasi OTP
3. Pilih paket (Uang/Sembako)
4. Tentukan target tabungan

#### Fitur Utama
- ✅ Catat setoran
- 📊 Pantau progress
- 🔔 Terima notifikasi
- 🎁 Pilih paket distribusi

### 2. Kolektor
#### Tugas Utama
- 📝 Input setoran peserta
- 🧾 Verifikasi bukti setoran
- 👥 Pantau wilayah

#### Fitur Khusus
- Scan/upload bukti setoran
- Lacak peserta belum setor
- Rekap setoran harian

### 3. Admin
#### Fungsi Manajemen
- 👤 Tambah/edit pengguna
- 🏦 Atur paket distribusi
- 📊 Laporan komprehensif
- 🔐 Kontrol sistem

## 🚀 Alur Penggunaan

### Peserta Baru
1. Buka aplikasi
2. Masukkan nomor HP
3. Terima OTP
4. Lengkapi profil
5. Pilih target tabungan
6. Mulai menabung

### Proses Nabung
- Setoran bisa via:
  * Kolektor (tunai)
  * Transfer bank
  * Pembayaran digital
- Minimal setoran: Rp 10.000
- Periode: Maret 2026 - Maret 2027

## 🎁 Pilihan Paket

### Uang Tunai
- Fleksibel
- Pilih nominal
- Transfer langsung

### Sembako
- Paket hemat
- Paket premium
- Sesuai kebutuhan keluarga

### Kombinasi
- Sebagian uang
- Sebagian sembako

## 📱 Cara Menggunakan

### Login

### Notifikasi
- Progress tabungan
- Pengingat setoran
- Info distribusi paket
