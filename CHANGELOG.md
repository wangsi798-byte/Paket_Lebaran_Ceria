# Changelog SiPaLe

## [1.0.0] - 2026-03-17 (Rilis Perdana)

### Fitur Utama
- ✨ Sistem tabungan paket Lebaran
- 👥 Manajemen peserta (admin, kolektor, peserta)
- 💰 Pencatatan setoran fleksibel
- 🎁 Distribusi paket (uang/sembako)
- 🔔 Notifikasi multi-channel
- 📊 Rekomendasi tabungan

### Keamanan
- 🔒 Autentikasi OTP
- 🛡️ Rate limiting
- 🔐 Enkripsi data
- 📝 Audit log

### Teknologi
- 🖥️ Backend: Node.js, Express
- 💾 Database: MongoDB
- 🌐 Frontend: React
- 🐳 Deployment: Docker

### Tambahan
- Dukungan multiple paket
- Visualisasi progress tabungan
- Manajemen stok paket

## Catatan Rilis
- Versi awal untuk wilayah kampung
- Fokus pada kemudahan penggunaan
- Aman dan transparan

- Integrasi pembayaran digital
